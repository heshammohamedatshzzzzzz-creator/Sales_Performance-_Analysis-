# Sales_Performance-_Analysis
•	Objective: Developed an end-to-end dashboard to analyze sales performance across both Retail and Wholesale sectors, tracking revenue, volume, and profitability.
•	Process: Integrated and cleaned datasets from different sales channels using Power Query. Created dynamic DAX measures to compare Wholesale vs. Retail trends and implemented hierarchical filters for product categories.
•	Outcome: Enabled stakeholders to identify high-margin wholesale clients and top-performing retail products, leading to better-informed inventory distribution and pricing strategies.
